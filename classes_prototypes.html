U jezicima koji koriste klase se klase definiraju prije rada s objektima, a objekti se instanciraju koristeći željene klase. Ako su dva objekta instancirana koristeći istu klasu, ta dva objekta u pravilu imaju identične varijable, konstante i funkcije te ih se može koristiti uporabom istog koda. Neki od popularnijih jezika koji koriste klase su: Java, C++, Python, C#, ...

U jezicima koji koriste prototipe klase ne postoje te se objekti direktno konstruiraju. Neki objekt se također može instancirati koristeći drugi objekt kao prototip pri čemu on nasljeđuje sve varijable, konstante i funkcije koje su deklarirane u originalnom objektu. Jezici koji koriste prototipe su obično jednostavniji za koristiti od jezika koji koriste klase ali su ujedno i manje sigurni. Primjeri jezika koji koriste prototipe su: JavaScript (i ostale ECMAScript implementacije), Lua, Io, ..,

Klasa (Kotlin)

class Pas(val boja: Boja = Boja.Crna) {
  var godine = 0

  fun ostari(brGodina: Int) {
    godine += brGodina
  }
}

object GlavnaKlasa {
  fun main(argumenti: Array) {
    val Madogga = Pas()
    Madogga.ostari(2)
  }
}

Prototip (JavaScript)

var pas = {boja: "Crna", godine: 0}

var madogga = Object.create(pas);
madogga.godine += 1;

madogga.boja; // "Crna"
madogga.godine; // 1
